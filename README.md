# java-web-gradle-jsp-hello-world

## Description
A POC for a jsp with java gradle build.

## Tech stack
- gradle
- bootstrap

## Docker stack
- gradle:4.2.1-jdk8-alpine

## Build notes
It takes about 1 minute for the server to be callable.

## To run
`sudo ./install.sh -u`
Available http://localhost/app/index.jsp

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
